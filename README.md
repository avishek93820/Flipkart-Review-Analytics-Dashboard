# Flipkart Review Analytics Dashboard

## Overview

This project analyzes 2,300+ Flipkart customer reviews using Python, SQL, NLP, and Power BI to generate business insights.

## Tech Stack

- Python
- Pandas
- NumPy
- SQL
- Power BI
- TextBlob

## Features

- Data Cleaning
- Sentiment Analysis
- Brand-wise Rating Analysis
- Interactive Power BI Dashboard
- KPI Cards
- Review Analytics

## Dashboard Preview

![Dashboard](Screenshots/dashboard.png)

## KPIs

- Total Reviews
- Average Rating
- Positive Reviews
- Negative Reviews
- Total Brands

## Dashboard Includes

- Rating Distribution
- Sentiment Split
- Average Rating by Brand
- Top Reviewed Brands
- Product Table
- Interactive Slicers
